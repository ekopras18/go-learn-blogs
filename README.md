## How to run

```bash
# Clone this repository
$ git clone https://github.com/ekopras18/go-learn-blogs.git

# Go into the repository
$ cd go-learn-blogs

# Install dependencies
$ go mod tidy

# Run
$ go run . 

# or Run with hot reload (install air first)
$ go install github.com/cosmtrek/air@latest

# then
$ air .

```
